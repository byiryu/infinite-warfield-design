재정초 배경. 27차 로고 4안(A 네온호라이즌·C 다크포지 네온림 등)은 네온 메타 톤 전제였다 → 밝은 쿨 D 확정으로 stale. 아래 4안 = 밝은 쿨 판타지 지휘소(라이트 스톤/하늘 + 골드) 톤. 아트 바이블 "전장은 어둡다 / 캐릭터는 빛난다" + 코어 테마(무한 전장·환생·5 원형 세력)를 어떻게 담는지가 축.

Concept A

∞ 인시그니아

무한대(∞) = 무한 전장 + 환생(차원의 법칙)을 한 마크로. 게임 코어 서사와 직결·기억성 최상. 워드마크는 Cinzel 하단 정렬.

강점: ∞ = 코어 테마 직결 · 앱 아이콘으로 완벽히 크롭 · 심볼 하나로 확장(로딩·워터마크)
약점: ∞ 자체는 게임 장르(전투/소환) 암시 약함 · 흔히 쓰이는 기호(차별화 = 포지 질감·색으로)
용도 = 앱 아이콘·인게임 타이틀·심볼 단독 · 아이콘 = 다크 배경 위 골드 ∞ (전장은 어둡다 + 빛나는 마크 = 아트 바이블)

Concept B

각인 크레스트 · 5세력

문장(heraldry) = 5 원형 세력의 통합·지휘소 위엄. 5분할에 테마색을 off-unit 액센트로만(도색 규칙 정합). 중앙 보스 = ∞. 골드 프레임.

강점: 5세력 세계관 즉시 전달 · 문장 = 전략/지휘 장르감 · 골드+스톤 = 밝은 쿨 정합
약점: 소형(아이콘)서 5분할 뭉갬 우려 · 요소 많음(단순화 필요) · 문장류 = 다소 흔함
용도 = 인게임 타이틀·상점/스토어 문장·로딩 엠블럼 · 아이콘 = 크레스트 단독(∞ 보스 강조)

Concept C · ★ 권고

2존 포지 art-bible 일체

로고 자체가 2존 대비를 형상화 — 위 "INFINITE"는 밝은 쿨(하늘 라이트, 골드 림) / 아래 "WARFIELD"는 어두운 전장 위 골드. 구 네온 림 → 골드 림으로 계승. "전장은 어둡다 / 위는 밝다"가 타이틀에서 바로 읽힘.

강점: 아트 바이블·2존 정체성 직접 인코딩(차별점) · 워드마크 중심 = 장르/이름 명확 · 인트로 시네마틱 강함
약점: 앱 아이콘 = 워드마크 크롭 애매(→ IW 모노그램/∞로 분리) · 2줄 = 가로 좁은 배너선 축소
용도 = 인게임 타이틀·인트로·메인 헤더 · 아이콘 = IW 모노그램(2존 split) 또는 A의 ∞ 조합 권장

Concept D

지평선 워필드

"Warfield"를 무한히 뻗는 전장 지평선으로 문자화 — 쿨 하늘 + 골드 지평선 + 소실점 원근(무한) + 떠오르는 ∞. 시네마틱 인트로 타이틀에 강함.

강점: "무한 전장" 문자 그대로 시각화 · 밝은 쿨 하늘 = 톤 정합 · 인트로 애니 여지(지평선 진입)
약점: 정보량 많아 소형서 약함 · 밝은 하늘 only = "전장은 어둡다" 결이 옅음(다크 전투 대비 약)
용도 = 인트로 시네마틱·메인 배경 통합 · 아이콘 = 지평선+∞ 배지
