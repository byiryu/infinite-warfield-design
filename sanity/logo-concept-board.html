Concept A · ★ 권고

다크 포지 · 각인 한글 패널 수렴

단조된 강철에 골드로 새긴 "전장" — 세력이 무너뜨릴 수 없는 영원한 전장의 기념비. 5명 중 4명이 이 방향으로 수렴.

강점: "전장" 2글자 위계 = 소형/앱아이콘까지 생존 · 강철+골드 각인 = 게임 무게·픽셀 블록 미학과 하모니 · 한글 주인공 = 유틸감 근원 제거 · 한국 시장 검증 문법
약점: 굵은 고딕은 흔함 → 각인 bevel·5세력 액센트·질감이 차별점을 만들어야(약하면 그냥 굵은 글씨). 세계관 서사는 은은(문장/씬보다 함축적)
담는 것 = 재질(강철 단조)·다크 정박·한글 임팩트 · 폰트 = Black Han Sans(개조) + Cinzel 라틴 kicker · 5세력 = 미니 sigil/색 액센트로 추가 가능
B레인 클래시 (Lane Clash) — 좌우 대치를 로고화로고 = 게임 폼
Concept B

레인 클래시 · 좌우 대치

좌우 두 진영이 중앙에서 부딪히는 순간 = 게임 코어를 로고가 그대로 시연. 아군 청색 림 ⊥ 적 적색 림(전투 실제 색 규칙 인용).

강점: 로고=게임 스크린샷 압축 → 장르 0.5초 즉독 · 다크+림라이트 = 아트 바이블 직역 · 스토어 CTR 유리
약점: 실루엣 많으면 소형서 복잡(좌1·우1 절제 필요) · 청/적 대립이 다소 일반적(세계관 신비감 약)
담는 것 = 코어 메커니즘·진영색·다크 · 실루엣 = 스키매틱(최종 = 실 픽셀 스프라이트·PixelLab) · 아이콘 = 중앙 충돌 크롭
C차원의 균열 (The Rift) — 무한·환생을 '차원'으로∞ 없이 무한 표현
Concept C

차원의 균열 · 환생 사이클

∞ 없이 "무한 전장·환생=차원 법칙"을 정면으로 — 다크를 가르는 균열, 새어나오는 골드 코어, 5세력 색, 흩어지는 픽셀 파티클(소환↔환생).

강점: ∞ 반려를 우회하면서 세계관 심장을 로고화 · "이 게임엔 세계관이 있다" 즉전달 · 다크 미스틱 밀착 · 소환/환생 게임필
약점: 밀도 높아 소형서 뭉갤 위험(아이콘=코어만 크롭) · 5색 과하면 산만
담는 것 = 세계관(무한 전장)·환생·5세력·다크 미스틱 · 글자 균열 통과 · 아이콘 = 세로 균열+코어 크롭
D각석 문장 (Crest) — 5세력·환생 엠블럼심볼형·앱아이콘 강함
Concept D

각석 문장 · 5세력 엠블럼

균열 간 방패에 5세력·중앙 코어(환생)·열린 룬 아크(재도약=닫힌 ∞ 아님). 문장 = 세계관 압축 + 심볼형이라 앱아이콘/브랜드 확장 최강.

강점: 심볼형 = 앱아이콘 소형 가독·브랜드 확장 최강 · 5세력 세계관 압축 · 다크+골드 = 톤 정합 · 열린 아크로 ∞ 없이 환생
약점: 플레이(레인배틀)를 직접 안 보여줘 장르 전달력 약 · 5분할 소형서 단순화 필요 · 문장류 다소 흔함
담는 것 = 5세력·환생·권위 · 아이콘 = 문장 단독(원생 심볼) · 아르덴 화염 대검을 중앙에 넣는 변주 가능
