확인 / 조정 모델

덱은 유지(persist) · 1탭 전투 시작

지난 전투 덱 그대로 → 적 위협 보고 바꾸고 싶을 때만 조정. 백지 재구성 강제 X = 반복 그라인딩 마찰 보호. 덱은 여기서만 짠다(메인 사전 편성 없음).

공간 멘탈 모델

좌 = 내 덱 · 우 = 적 (전투와 일치)

전투의 아군 좌·적 우와 좌우 정합. 덱 영역 우선(~62%·주 인터랙션·슬롯 크게) / 적 진형 = 참조(~38% 컴팩트).

로스터

덱 하단 2줄 가로 스크롤

보유 37유닛 → 1줄은 스크롤 과다 → 2줄. 탭 → 빈 슬롯 편입 · 신규 해금 NEW · 미해금 🔒 · 편성된 유닛 dim. 별도 도감 화면 없음.

위협 텔레그래프 (CEO Q5)

적 진형 프리뷰 + 2축 언어

적 편성 위치 상설 프리뷰 + 위협 아키타입 라벨. 2축 언어("중장갑/마법취약") + 응수 hint("→ 마법 딜러 추천") + 카운터 테마(5테마 색). 팝업 → 상설 격상.

난전 가독 · 림라이트 (P0 게이트)

아군 파랑 / 적 빨강

편성부터 전투까지 일관된 진영 식별. 유닛은 다크 인셋에 거주(밝은 크롬 위 픽셀 배치 X) = 가독 보호. 테마색 = off-unit 신호(테마칩·위협·카운터).

터치 인체공학 (paranoia #3)

전투 시작 = 우하단 엄지존

"덱 빌드 → 적 확인 → 시작" 오른손 흐름. 취소 = 전투 시작 위(작게·점선 = 미스탭 가드) ⊥ 전투 시작(크게·골드 글로우·펄스 = 1차 위계). 덱 슬롯·로스터 카드 ≥44pt.
