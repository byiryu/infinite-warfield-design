판정 + 디자인 권고사용자 확정

Q1 실루엣 충분한가? (A) — 발톱/골렘/잎/해골/방패 모티프로 진영이 읽히는지. 디자인 관찰: form-정렬 best-case라도 Combat·Necro 둘 다 어두워 실루엣만으론 약함.

Q2 off-unit 베이스 링 채택? (B) — gdd 완충. 권고: 채택. 색을 유닛에 안 칠하고도 진영 즉독·특히 cross-form(form≠진영) 유닛에 필수. 단 난전서 링 과밀 주의.

Q3 림라이트 필수? (C) — 권고: 거의 필수. 5종 다 어두움 → 다크 전장 figure-ground 분리(gdd "캐릭터는 빛난다"·Necro 림라이트 강제 정합). 실제 구현 = 스프라이트 외곽 라이트/셰이더(plat) 또는 생성 시 밝은 외곽.

✅ 확정 (2026-06-30 사용자): C 림라이트 채택 = 아군/적 색(아군 #4FA3FF / 적 #FF5B5B — 전투 식별 + 가시성을 한 신호로). 테마 진영색·발밑 링 = 전투 화면 제외 → 편성·유닛 상세·위협 텔레그래프 등 계획(planning) UI 전용(전투선 actionable 아닌 클러터 회피·카운터 판단=계획 화면). 실루엣 = 보조. → gdd "테마색 off-unit" 원칙 유지(본체 도색 X·off-unit 위치를 계획 UI로 집중). gdd Part 3.2 "(권장)베이스 링" 정합 = PM escalate. 37 양산 룩 = 락(전투 = 아군/적 림 + 실루엣 + 자연 팔레트).
