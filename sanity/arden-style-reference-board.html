아르덴 스타일 레퍼런스 — 치비 톤 결정

프로세스 리셋: 컨셉(1) → 레퍼런스(2·여기) → 스타일 스펙(3) → 생성(4). 그동안 1~3 없이 4부터 해서 빗나갔음. 이제 톤 타겟을 정하고 생성한다.

못 박은 컨셉 (인터뷰)
아르덴 = 백전노장 사령관 (수많은 전장을 버텨낸 노련함·수염·상처·중후한 갑옷) + 거대 화염 대검(시그니처·한눈에 식별) · 마젠타 화염 = Combat 신호 · 자가증폭("버틸수록 강해짐") · Combat 영웅.
치비/SD = 큰 머리(전체 1/3~1/2)·2~2.5등신·작은 몸. 목표 = 진짜 치비 + 매력 둘 다.

A. 레트로 픽셀 SD (치비 도트)

매력적 픽셀 치비의 실증. CraftPix "Knight Chibi" = 빛나는 눈·또렷한 컨투어·균형 잡힌 치비 비율의 애니메이션 도트 기사. = 픽셀로도 매력적 치비가 된다는 증거. 우리 매체(PixelLab) 네이티브 = 매체 변경 0. 백전노장+화염검을 이 톤으로 타겟하면 됨.

CraftPix Knight Chibi ↗ Frost Knight Chibi ↗ OpenGameArt Chibi heroes ↗ ArtStation Chibi Knight ↗
🟡 일러스트/3D = 매체 escalation

B. 클래시로얄 볼드 치비

굵은 아웃라인·선명한 색·볼드한 3D→2D 치비. 난전 가독 골드스탠다드·흥행 캐주얼 대표. 단 pixel 아님(3D 렌더·일러스트) → 이 룩을 원하면 매체를 pixel→일러스트로 바꾸는 리드 escalation(2026-06-29 pixel 확정 재오픈).

핵심 인사이트
"치비 = 밋밋"은 매체 탓이 아니라 스타일 타겟 부재 탓이었다. CraftPix Knight Chibi(A)가 증명 — 픽셀로도 매력적 치비가 됨(빛나는 눈·또렷한 라인·디테일). 이전 standard 렌더가 밋밋했던 건 명확한 스타일 레퍼런스 없이 뽑아서야. → A(레트로 픽셀 SD)를 타겟으로 스타일 스펙(라인·등신·디테일·팔레트)을 잡고 큰 size로 재생성하면 매체 유지하며 매력 확보 가능성.
Clash/Rumble(B/C) 수준의 "일러스트 매력"을 원하면 = pixel 포기 = 리드 매체 escalation.
결정 / 다음방향 → 3단계 스타일 스펙
  • 권고 = A(레트로 픽셀 SD) 타겟·매체 유지. 매력적 픽셀 치비가 가능함이 증명됨(CraftPix). A를 타겟으로 스타일 스펙 잡고 재생성 → 충분히 매력적이면 매체 escalation 회피.
  • 또는 B/C(일러스트·3D룩) 원함 = 매체 escalation. Clash/Rumble 수준 매력이 필수면 pixel→일러스트 전환을 리드/CEO에 escalate(2026-06-29 재오픈). 솔로 production 경로 비용 동반.
  • 레퍼런스 더 필요? 특정 게임/캐릭터를 콕 집어주면 그 URL·분석 추가.

방향 정하면 → 3단계 스타일 스펙(등신·라인 두께·디테일·팔레트·size) 문서화 → 그 스펙으로 아르덴(백전노장+화염검) 재생성.
