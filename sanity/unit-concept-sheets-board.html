5진영 일반 유닛 — 컨셉 시트

난전 진영 가독 P0 검증의 step 1 (컨셉). GDD가 확정한 "테마색 = off-unit 신호 전용" 베팅을 실제 스프라이트로 검증하기 위해, 진영별 대표 로스터 유닛 1기씩의 정체성·실루엣·시그니처를 락한다.

● 진행 중 · 컨셉 단계 — 사용자 검토 → 컨셉아트 생성 핸드오프

왜 — GDD 새 베팅 · 게이트gdd Part 3.2

GDD Part 3.2 (CEO 재정의): 테마색은 off-unit 신호 전용(UI·범례·텔레그래프·아군/적군·베이스 링). 유닛 본체 도색 ≠ 테마색 — 유닛은 자연 팔레트로 그리고, 난전 진영 식별의 1차 부담은 실루엣 + 모티프가 진다. GDD 본문이 직접 리스크 표시 = ~2.5등신 치비·5진영에서 실루엣만으로 즉독은 난도 높음 → 실 스프라이트로 검증(가정 금지).

이 P0가 답하는 질문:

A · 베이스 링 없이 실루엣+모티프만으로 난전(~32px)에서 진영 5종을 즉독할 수 있나?
B · off-unit 진영색 베이스 링/그라운드 데칼 추가 시 개선폭은?
→ 판정: 실루엣 충분한가 · 베이스 링 채택할까 → design-system §1 베이스 링 결정 + 37체 양산 룩 확정.

★ 핵심 인사이트 — form ≠ 진영units.json

로스터의 form(시각 체형: Human/Undead/Spirit/Machine/Beast)이 unitType(진영)과 자주 어긋난다 — 예: Combat 전투 저격수 form=Machine · Spirit 생명의 고리 form=Beast · Machine 강철 하운드 form=Beast(동물형 기계).

함의: 실루엣은 form을 따른다 → 같은 진영 안에서도 체형이 섞이면 실루엣만으로 진영을 못 읽는 경우가 구조적으로 존재. = off-unit 베이스 링이 필요할 가능성을 이미 시사.

이번 P0 = best-case 테스트: 아래 5체는 전부 form이 진영에 정렬(Combat=Human · Necro=Undead · Spirit=Spirit · Machine=Machine · Beast=Beast)된 cost-4 근접 유닛. 실루엣 가설의 가장 유리한 조건에서 검증 → 여기서도 실루엣이 부족하면 베이스 링은 필수. 통과하면 다음은 cross-form 유닛(베이스 링이 진영을 carry)을 별도 검증.

★ 공통 생성 가드 (사용자 피드백 2026-06-30)전 유닛·전 프롬프트 적용

① 배경 제거 — isolated 캐릭터: 컨셉아트/스프라이트는 투명·단색 배경의 고립된 캐릭터여야 한다(전장 배경 X). 다크 전장은 plat이 인게임에서 렌더 — 스프라이트에 포함하면 안 됨. 전 프롬프트 "dark battlefield backdrop" → "isolated on plain transparent background, no scene".

② 영웅 차별화 — 일반 유닛 ≠ 영웅 (gdd Part 3.1): 영웅 distinction(전용 아우라·망토·과대 시그니처 무기·크기 +20~40%·림라이트)은 영웅 예약. 일반 유닛은 졸병/정규군 톤 = 아우라 X · 망토 X · 과장/화염 무기 X · 수수한 장비. ★ 발견 (사용자) Combat 졸병 첫 생성이 영웅 아르덴(불검·크림슨 망토·뿔투구·마젠타 아우라)과 중복 → 화염 무기/망토/특수 아우라/대검 전면 금지로 수정. (Necro 일반 ↔ 베스라 보라 orbit · Spirit 일반 ↔ 에이라 에메랄드 링도 동일 가드.)

③ 본체 테마색 도색 X (gdd Part 3.2): 진영색은 off-unit(베이스 링/UI)만 — 본체는 자연/컨셉 팔레트.

스코프 결정 · 워크플로우아르덴 동일

스코프 = 정적 5체 먼저, 애니는 게이트 통과 + 룩 승인 후 (권고). 이유: 애니는 유닛당 6종 × 5체 = ~30 gen + 멀티턴이라, 룩이 바뀌면 매몰. 아르덴이 이미 애니 파이프라인(생성→MCP 6애니→export)을 ✅ 증명했으니 5체에 재증명 불요. 먼저 "이 5진영 룩이 난전서 읽히나"(게이트)를 통과시키고, 그 룩으로만 애니에 투자.
