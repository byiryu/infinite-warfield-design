③설계 근거 · A/B 판단 가이드
A안 = 라이브 일러 (2존 정합)

META 존 = painterly 일러 언어

일러 포트레이트 = 35차 확정("영웅 일러 포트레이트는 편성·선택 화면에") + 장르 표준(AFK 등 일러+SD 인게임). 라이브 = 정지 일러 + 오버레이만(브리딩 스케일 1.025·테마 글로우 맥동·잔불/오브/위습 파티클) = 메인 허브 라이브 배경과 동일 원리·Unity 파티클/셰이더 무료·영상 아님. 대형 사이즈 감정 무게(누구랑 시작할까) = 일러의 일.

B안 = 픽셀 유닛 (전투 정체성)

"고르는 것 = 전장에서 싸울 그 모습"

선택 화면부터 인게임 실물 = 충실도 갭 0. 단 32~48px 스프라이트를 카드 크기로 확대 = 정보 밀도 낮음·감정 무게 약함. ★ 베스라/에이라 = 일반 유닛 스탠드인(영웅 픽셀 = 아르덴 Unity 게이트 후 생산) — 룩 방향 판단용.
