● B 방향 선택 → 실 캐릭터 조합 · 앱아이콘 = 캐릭터 클로즈업 · 검토 대기

v3 보드의 좌우 실루엣은 스키매틱 플레이스홀더였습니다. 아래는 실제 생성된 픽셀 SD 캐릭터(아르덴 Combat 영웅 · Necro 혈귀)를 그대로 넣은 진짜 목업 — 아군 파랑 림 ⊥ 적 빨강 림 + 골드 충돌 = 전투 화면 실제 규칙 그대로. "이게 곧 게임"이 보드에서 확인됨.

✓ 진짜 (실 에셋)

  • 캐릭터 = 실제 생성 픽셀 SD 스프라이트 (아르덴 char 2b180175 · Necro 359fdc4a) — 코드 도형 아님(매체게이트 #8 준수)
  • 아군 파랑 / 적 빨강 림 = 전투 화면 P0 게이트 확정 규칙 그대로 인용
  • 다크 전장 정박 + 골드 충돌 = 아트 바이블 "전장은 어둡다/빛난다"
  • "전장" 대형 + "무한의" 소형 = 패널 수렴 위계

◆ 튜닝 가능 (정밀화 시)

  • 대진 매치업 = 아르덴 vs Necro는 예시 · Beast/Machine 등 교체 가능(가장 "센" 조합 택)
  • 포즈 = 정지 대신 attack 프레임(검 스윙·단검 찌르기)로 에너지 ↑ 가능
  • 충돌 이펙트 = 골드 스파크 정밀화(불티·검격 라인)
  • 아이콘 = 전용 고해상 포트레이트 별도 생성(임시 크롭 아님)
  • 유닛 수 = 1v1(로고 정석) / 뒤에 소형 실루엣 군세 추가 여부
