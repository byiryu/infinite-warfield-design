왜 이 파일럿? 메인 허브 목업이 플랫 CSS(매끈 벡터)라 픽셀 게임과 재질이 안 닿는다는 지적 → PixelLab로 실제 픽셀 UI 킷을 뽑아 스타일 검증. 아래가 그 실물이다 (밝은 쿨 판타지 지휘소 = 쿨 스톤 + 골드/브론즈 각인 + 젬 액센트).

PixelLab 생성 · 512×384 · 버튼(젬 액센트) + 각인 패널 프레임 + 사이드 아이콘 툴바 · 영문 라벨(LOAD/NEW/COMMAND)은 PixelLab 기본 더미 = 실제는 한글·우리 라벨로 교체

✓ 검증된 것 (스타일 방향)

  • 재질 일치 — 픽셀아트 = 게임(픽셀 SD 유닛)과 한 세계로 읽힘 (플랫 CSS의 이물감 해소)
  • 밝은 쿨 + 골드/브론즈 = design-system 톤 그대로·"지휘소" 위엄
  • 버튼(젬 액센트) · 각인 패널 프레임 · 아이콘 툴바 = 하이브리드 킷 요소 확보
  • 가독성 = 클린 레이아웃 위에 픽셀 프레임 = 하이브리드 성립

◆ 다음 (프로덕션 화)

  • 개별 프로덕션 에셋 = 9-slice 패널 프레임 · 주 CTA(골드) 버튼 · 보조 버튼 · 재화/메뉴 아이콘 — 우리 규격/한글로 재생성
  • 실제 하이브리드 메인 허브 = 이 픽셀 프레임/버튼 + 실 캐릭터(아르덴/Necro) + 클린 레이아웃 조합
  • 톤 미세조정: 더 밝게 / 골드 양 / 각인 강도 = 실물 보고
