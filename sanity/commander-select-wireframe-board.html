#3 터치 인체공학 ✅ 카드 = 큰 탭 타깃(≥120px 폭). [시작] CTA = 중앙 하단 엄지존 · 44pt↑. 가로 양손 파지 = 카드 좌우 탐색 자연.
#8 매체게이트 ⛔ ⚠️ 영웅 아트 = placeholder(점선 박스 + 이모지 + 테마 아우라). appeal 판정 X = 픽셀 매체 확정(CEO 2026-06-29) · PixelLab 생성(아르덴 P0 = 정본 확정·이하 베스라/에이라 후속). 본 보드 = 카드 위계·존·CTA·글로우 = 레이아웃 도메인만.
정보 기반 선택 ✅ 추천·강제 없음 = 3종 동등 제시. 대신 counter 강/약을 카드에 노출 → 첫 선택부터 "내 영웅이 무엇에 강/약한가" 멘탈 모델 형성(counter-deck 학습 시작점).
