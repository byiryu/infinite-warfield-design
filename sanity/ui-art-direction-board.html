◆진단 — 왜 2안이 정체성이 없었나
근본 원인 = 재질(플랫/픽셀)이 아니라 "세계관 부재". 두 시안 다 정체성을 가장 얕은 레이어(색·재질)에서만 만졌고, 정작 정체성이 사는 곳(모티프·프레임 은유·분위기)이 비었다. 플랫=스톡 하늘+둥근 흰 카드=SaaS 유틸 문법 / 픽셀=남의 PixelLab 킷(LOAD/NEW 더미)=원천적 무명. 경쟁작 통찰: 흥행 UI가 제네릭을 탈출한 유일한 법 = 세계관 로어에서 재질을 역산(AFK=선술집 목재·오딘=바이킹 룬석). → "무한 전장은 어떤 물질로 이뤄졌나?" = 차가운 룬각석(밝은 쿨 스톤).
★확정 제안 — 무한의 전장 UI 정체성4-에이전트 종합

"차원 관문 앞의 작전 지휘소 — 밝은 룬각석 콘솔 위에서, 아래로는 어두운 무한 전장이 영원히 타오른다."

재질 = 페인터리 일러스트 위 차가운 룬각석 + 브론즈 룬 상감 + 홈 발광(무한 차원의 물질). 팔레트 = design-system §1 밝은 쿨 + 골드/브론즈. 인게임 픽셀 유닛은 다크 전투 인셋에 거주(2존 대비의 일부).

① 무한전장·차원균열·룬 = 배경/프레임 언어(예쁜 하늘 금지·끝없는 지평/균열/룬 관문)
② 5세력 = 문장(crest)+재질 모티프 (색 라벨 dot 아님 — 나란히 놓으면 색 빼도 구별)
③ 2존 = 재질 대비 + 한 화면 다크 인셋 (밝은 콘솔에 어두운 전장 창이 뚫림·격리 금지)
④ 지휘소 은유 = 전황 테이블·차원 관문·각석 콘솔 (버튼/카드 아님 = 석판/관문/계기판)
Combat · 교차검/방패 · 각진 강철 Necro · 두개골/낫 · 부식 브론즈 Spirit · 잎/룬고리 · 옥·수정 Machine · 톱니/코어 · 풍화금속(NOT 네온) Beast · 발톱/송곳니 · 거친 뼈·가죽
★ ∞ 기호는 로고서 폐기(범주 오류) → UI 서명 형상 = 룬 관문/차원 균열/각석 (∞ 아님). Machine = Horizon 로스트테크(풍화 금속+코어), 네온 사이버 아님.
▣3방향 + 바로 쓸 Gemini/Midjourney 프롬프트프레임 은유가 다름

셋 다 위 정체성(룬각석·5세력 문장·2존 다크 인셋·∞없는 무한)을 공유하되 지휘소 프레임 은유가 다르다. 아래 프롬프트를 Gemini/Midjourney에 그대로 붙여 각 1~2장 생성 → 실물로 비교(매체게이트 #8 = appeal은 실렌더 판정).

방향 1 · ★ 권고 (paranoia #1 위계 최강)

차원 관문 지휘소 (Threshold Gate) 2존 서사화

화면 전체가 거대 룬 관문 아치 = UI 프레임. 문 너머 밝은 쿨 하늘 + 어두운 무한 전장. "밝은 관문 ↔ 다크 전장"이 프레임 자체.

  • 프레임 = 룬각석+브론즈 관문 아치(키스톤·좌우 문장 필라·문지방 CTA)
  • 무한/환생 = 포털 너머 소실점 무한 전장 + 아치 둘레 룬 순환(환생=관문 재진입)
  • 2존 = 밝은 석재 아치 ⊥ 포털 안 다크 전장(#0D1117·아군청/적적 림)

방향 2 · 전략·중후 (메뉴 밀도 강)

마법 전황 테이블 (Arcane War Table)

밝은 석조 지휘실의 대형 원형 전황 테이블 위로 무한 차원 지도가 살아 움직인다. 룬 서클 환생. 손에 잡히는 전략감.

  • 프레임 = 밝은 룬석 지휘실 + 원형 전황 테이블(하이앵글)·황동 기둥·아치창 쿨광
  • 무한/환생 = 경계 없는 안개 지도(무한) + 중앙 룬 서클(쓰러진 말 정화 재배치=환생)
  • 2존 = 밝은 테이블 ⊥ 위협 프리뷰 다크 인셋

방향 3 · 재질 앵커 (경쟁작 #1 추천)

각인된 전장 콘솔 (Graven Warfield Console)

UI = 무한 차원의 물질인 룬각석 슬랩으로 만든 유물 지휘 콘솔. 음각 홈 골드 발광·승리를 돌에 새김. 재질 충실도 최강.

  • 프레임 = 풍화 밝은 룬석 슬랩 + 브론즈 트림 + 음각 룬(홈 안에서만 발광=밝은 표면 유지)
  • 무한/환생 = 콘솔에 뚫린 창으로 아래 끝없는 다크 전장·환생=새 균열 개방
  • 2존 = 밝은 각석 콘솔 위 ⊥ 창 아래 어두운 무한 전장
